# 原料索引

## 使用方式
- 文章、网页、访谈、视频字幕、报告先放这里。
- 原料不追求完美总结，只保留来源和关键事实。
- 真正重要的东西再提炼到 [[_概念地图]]。

## 最近原料
```dataview
TABLE status, domain, source, file.mtime AS "更新"
FROM "10_原料库"
WHERE file.name != "_原料索引"
SORT file.mtime DESC
LIMIT 50
```
