# 下周启动手册

按当前环境时区 Asia/Shanghai 计算，今天是 2026-06-22，下周一是 2026-06-29。目标不是一次搭完复杂系统，而是先连续用 7 天。

## 第一次打开
1. 打开 [[私有Wiki首页]]。
2. 确认飞书机器人 `学习库收件箱` 可以收到消息。
3. 把今天看到的文章、观点、截图、灵感先转发给飞书机器人。
4. 晚上打开 [[私有Wiki首页]]，查看“每日机会雷达”。
5. 只从雷达里挑 1 条，决定保留、提炼或输出。

## 每天 10 分钟
1. 2 分钟：把今天看到的文章、观点、截图、灵感转发给飞书机器人。
2. 3 分钟：查看 `每日五领域信息候选`，勾选少量 `收录` 或 `移除`。
3. 3 分钟：查看 `每日机会雷达`，挑 1 条最有价值的内容。
4. 2 分钟：把它连到至少一个主轴：互联网、AI、游戏、投资、写作。

## 每周 30 分钟
1. 用 `周复盘` 模板新建一篇复盘。
2. 从本周输入里挑 3 条值得保留的原料。
3. 从原料里提炼 3 张概念卡。
4. 从灵感池里选 1 个推进成输出。
5. 投资方向必须补 `事实、假设、反证、触发条件`。
6. 写作方向必须补 `读者、观点、结构、标题实验`。

## 五个主轴怎么用
- 互联网：记录产品、平台、社区、增长、商业化和趋势判断。
- AI：记录工具、自动化、模型能力边界、AI 产品机会。
- 游戏：记录机制、系统、叙事、关卡、玩家体验。
- 投资：记录行业、公司、财报、交易复盘、风险信号。
- 写作：记录选题、结构、表达、标题、发布反馈。

## 插件分工
- QuickAdd：快速创建原料、概念、灵感、投资观察、写作卡片。
- Notebook Navigator：作为主文件浏览器，用文件夹和标签快速定位。
- Claudian：让 AI 在库里协助整理、归纳、重写和批量处理。
- Dataview：让索引页自动列出最近原料、概念、输出和观察。
- Templater：管理模板，减少重复格式输入。
- Tasks：把笔记里的 `- [ ]` 自动聚合成任务列表。
- Knomo：快速把零散备忘写进每日笔记。
- Latex Suite：写数学公式、投资模型或技术推导时使用。
- Excalidraw：画思维导图、系统图、产品结构和文章结构。
- Git：把知识库备份到 GitHub，并驱动公开网站构建。

## 手机使用原则
- 手机只负责把内容转发给飞书机器人，不做大规模整理。
- 桌面端负责周复盘、结构调整、批量整理和公开发布。
- 同步首选 Obsidian Mobile + Obsidian Sync；Git 同步适合桌面端，不建议作为手机新手入口。
- 公开查询使用 GitHub Pages 网站，只放 `public/` 里的公开内容。

## 每日自动筛选
- `Zen Daily Domain Brief` 每天 08:30 生成五领域信息候选。
- 你只需要在候选页勾选 `收录` 或 `移除`，不确定就空着。
- `Zen Daily Feedback Learning` 每天 23:10 学习你的勾选，沉淀来源和关键词偏好。
- 投资候选只用于观察和反证，不直接形成买卖建议。
