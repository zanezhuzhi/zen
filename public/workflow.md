# 知识工作流

## 最小闭环

```text
收集 -> 原料 -> 概念 -> 连接 -> 输出
```

## 每天

- 把微信、飞书、网页和临时想法放进收件箱。
- 清理至少 1 条收件箱内容。
- 有启发就提炼成概念卡片，不只保存原文。

## 每周

- 新建或更新 3 张概念卡。
- 把概念连接到互联网、AI、游戏、投资、写作五个主轴。
- 从灵感池里挑 1 个推进成文章、报告、产品想法、游戏设计或投资复盘。

## 公开发布原则

- 只发布已经整理过、适合外部阅读的内容。
- 投资内容记录事实、假设、触发条件和反证信号。
- 写作内容记录选题来源、文章结构、核心表达和发布反馈。
- 不发布密钥、日志、私密资料、未授权内容和完整转载材料。
