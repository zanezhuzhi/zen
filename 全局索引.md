# 全局索引

## 按类型
```dataview
TABLE type, status, domain, file.mtime AS "更新"
FROM "10_原料库" OR "20_概念库" OR "40_输出库"
WHERE type
SORT type ASC, file.mtime DESC
```

## 原料
```dataview
TABLE status, domain, source, file.mtime AS "更新"
FROM "10_原料库"
WHERE type = "source"
SORT file.mtime DESC
LIMIT 50
```

## 概念
```dataview
TABLE status, domain, file.mtime AS "更新"
FROM "20_概念库"
WHERE type = "concept"
SORT domain ASC, file.mtime DESC
LIMIT 80
```

## 灵感与写作
```dataview
TABLE type, status, domain, audience, file.mtime AS "更新"
FROM "40_输出库" OR "20_概念库"
WHERE type = "idea" OR type = "writing" OR contains(tags, "writing") OR contains(tags, "idea")
SORT file.mtime DESC
LIMIT 80
```

## 投资观察
```dataview
TABLE status, target, file.mtime AS "更新"
FROM "10_原料库" OR "20_概念库" OR "40_输出库"
WHERE type = "observation" OR domain = "投资" OR contains(tags, "investment")
SORT file.mtime DESC
LIMIT 80
```

## 每日机会雷达
```dataview
TABLE date, status, file.mtime AS "生成时间"

```

## 每日五领域信息候选

TABLE date, status, file.mtime AS "生成时间"

## 未分类内容
```dataview
TABLE type, status, file.folder, file.mtime AS "更新"
FROM "10_原料库" OR "20_概念库" OR "40_输出库"

```
